TASK: To determine whether or not a number is prime.

PYTHON CODE:

import math

def isPrime(x):
  for i in range(1, int(math.sqrt(x)) + 1):
    if x % i == 0:
      print("Composite")
      return
    else:
      pass
  print("Prime")
